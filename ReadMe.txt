制作物についての仕様
・クライアントからサーバーに文字やテキストファイルを送り、サーバーが送られてきたものを表示。
・テキストファイルの送受信についてですが、送れてはいますがサーバーの受信部分が反応しないため、完成できていません。

操作マニュアル
・サーバー、クライアントを実行
・サーバー、クライアント両方に文字を送りたい場合は1、テキストファイルを送りたい場合は2を入力。
・1、2以外を入力した場合（サーバーでは終了処理、クライアントではもう一度入力処理が呼ばれます。）
・1を入力　クライアントで送りたい文字を入力=>サーバーに送られ表示。
・2を入力　クライアントで送りたいテキストファイル名を入力(送りたいテキストファイルはプロジェクトファイル内に入れてください)=>サーバーが受信(されない)、ファイルの内容を表示(されない)。
